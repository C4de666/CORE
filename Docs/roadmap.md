# CORE Roadmap

CORE is built in public and evolves in stages.

## Phase 1 — Foundations (current)
- Stable taxonomy
- Beginner core notes
- MkDocs site online

## Phase 2 — Expansion
- Intermediate systems content
- Hands-on labs
- Contributor onboarding

## Phase 3 — Depth
- Advanced security, cloud, and AI
- Community curation
- Quality scoring and reviews
